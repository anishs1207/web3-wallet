# Contributing to Vaulta 🌌

Thank you for your interest in contributing to Vaulta! We welcome contributions from the community to help make this web-based Web3 wallet even better.

## 🚀 How Can I Contribute?

### Reporting Bugs 🐛
If you find a bug, please open an issue on GitHub and include:
- A clear description of the bug.
- Steps to reproduce the issue.
- Your browser and OS versions.
- Any relevant screenshots or error logs.

### Suggesting Features ✨
We love hearing new ideas! To suggest a feature:
- Open an issue with the "feature request" tag.
- Explain why the feature would be useful and how it should work.

### Pull Requests (PRs) 🛠️
1. **Fork the repository** and create your branch from `main`.
2. **Install dependencies**: `npm install`.
3. **Make your changes**: Ensure your code follows the project's style and conventions.
4. **Test your changes**: Run `npm run dev` and verify everything works as expected.
5. **Lint your code**: Run `npm run lint` to catch any consistency issues.
6. **Submit a PR**: Provide a clear description of what you've done and link to any related issues.

## 🛠️ Development Setup

Vaulta is built with:
- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **Solana & Ethereum Web3 Libraries**

To start developing locally:
```bash
git clone https://github.com/anishs1207/web-wallet.git
cd web-wallet
npm install
npm run dev
```

## 📜 Code of Conduct

Help us keep Vaulta a welcoming and inclusive project. Please be respectful and professional in all interactions.

## 🛡️ Security

If you discover a security vulnerability, please do NOT open a public issue. Instead, contact the maintainer directly or follow the project's security reporting process.
